# 💰 Loan Status Prediction Using Machine Learning

## 📌 About
A machine learning model that predicts whether a loan 
application will be approved or rejected based on 
applicant details such as income, credit history, 
and employment status.

## 🛠️ Tools & Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Seaborn
- Matplotlib

## 🤖 Models Used
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVC)
- Logistic Regression

## 📊 Results
| Model | Accuracy |
|-------|----------|
| Random Forest | 82.5% |
| Logistic Regression | 80.4% |
| SVC | 69.2% |
| KNN | 63.7% |

✅ Best Model: **Random Forest with 82.5% accuracy**

## 🔄 Process
1. Data cleaning and missing value handling
2. Label encoding for categorical variables
3. Exploratory Data Analysis (EDA)
4. Model training and comparison
5. Hyperparameter tuning

## 🚀 How to Run
1. Clone the repository
2. Install dependencies: 
   pip install -r requirements.txt
3. Run the notebook:
   loan_status_prediction.ipynb
